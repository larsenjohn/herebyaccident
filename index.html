Github Pages Blog Template

A Stripped down framework with the basic liquid markup for starting a jekyll site on github pages. Bootstrap 3.0, retina support.

The basics

The basic library is here, with general front matter info and structure.

You might want to watch out for the "baseurl", or "/repository-name" for links if you are planning to use use github without a custom url.

Otherwise configure it with a CNAME and point records over.

Markdown

Github pages 3.0 now only supports kramdown. Save file as a .md or .markdown file.

You'll proabably need to keep the reference in the config_yaml (markdown: kramdown).

Typically you would use markdown for blog posts, or something with a lot of general text. Basic markdown is included in atom, but you can also download the atom extension.
